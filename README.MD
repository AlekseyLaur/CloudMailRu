# Плагин [Total Commander](http://www.ghisler.com/) для работы с [cloud.mail.ru](http://cloud.mail.ru)

### Описание
Плагин в процессе приготовления. В настоящий момент поддерживаются:
* Работа с несколькими аккаунтами
* Навигация по каталогам
* Скачивание файлов из облака

Реализован только Unicode-набор функций, на старых версиях TC плагин работать не будет.
Поддержка x64-версии TC имеется.

Перед использованием ознакомьтесь с [лицензионным соглашением по использованию Сервиса Облако@mail.ru](https://cloud.mail.ru/LA/)

### Установка и использование
Склонируйте или скачайте репозиторий на диск, и подключите плагин MailRuCloud.wfx/MailRuCloud.wfx64. Создайте в каталоге плагина файл MailRuCloud.ini с содержимым вида

[Название аккаунта]
user=логин аккаунта без указания домена
domain=домен аккаунта
password=пароль аккаунта

Соответствующий аккаунт добавится в плагин. Количество аккаунтов неограничено, просто добавляйте секции ini-файла по образцу.
